Here are some Notion-related projects and resources from enthusiastic people. There may be more, I will update the list once I discover them!

API Clients / Wrappers / Libraries

  • kjk/notionapi - Unofficial Go API for Notion.so
    Using Notion API Go client

    Content of Notion consists of pages. Each page has a unique id. The id is the last part of Notion URL. For example page https://www.notion.so/Test-page-all-c969c9455d7c4dd79c7f860f3ace6429 has id c969c9455d7c4dd79c7f860f3ace6429.

    https://presstige.io/p/Using-Notion-API-Go-client-2567fcfa8f7a4ed4bdf6f6ec9298d34a

  • jamalex/notion-py - Unofficial Python API client for Notion.so
    Introducing notion-py, an unofficial Python API wrapper for Notion.so

    Hello! My name is Jamie Alexandre, and I'm an API addict. I'm always hesitant to pick up a new tool if it doesn't have some way for me to interact with it in my native language: code. APIs are the...

    https://medium.com/@jamiealexandre/introducing-notion-py-an-unofficial-python-api-wrapper-for-notion-so-603700f92369

  • petersamokhin/knotion-api - Unofficial Notion.so API wrapper, written in Kotlin
  • cstrnt/notion-api - Unoffical Notion.so API (Node.js)
  • sorcererxw/notink - Notion.so API [WIP] (Node.js)
  • dragonman225/notionapi-agent - Unofficial Node.js API client for Notion.so (made by me)
  • mayneyao/notabase - A JavaScript library to manipulate Notion's database blocks.

Applications Using Notion's API

  • mayneyao/gine-blog - A blog built with Notion's API and Gatsby, React, Material-UI, Netlify.
  • shariq/notion-on-firebase - Serve notion.so pages on Firebase hosting.
  • sorcererxw/blog - A blog built with Notion's API.
  • mayneyao/NotionPlus - Code in Notion, run on self-hosted server, and display result in Notion.
  • dragonman225/notablog - Generate a minimalistic blog from a Notion.so table. (made by me)
  • Presstige.io - Publish Notion pages on the web.
  • GoodWrites - Turn your Notion documents into a blog or web site.
    Luke's Website

    A blog about building GoodWrites, Ruby on Rails, Technology and sometimes sports

    http://luke.goodwrites.co/

  • Cobertos/md2notion - A better Notion.so Markdown importer.
  • ijjk/notion-blog - A Next.js site using new SSG support with a Notion backed blog.
    My Notion Blog

    An example Next.js site using Notion for the blog

    https://notion-blog.now.sh/

  • heycalmdown/notion-mermaid - Draw mermaid diagrams in Notion.
  • yannick-cw/notion-ocr - Adding OCR support to Notion.

Community

  • Notion Hacks - A place / community to collect, discuss and create various hacks for Notion.
  • Notion on Crowdcast - Notion office hours hosted by Marie Poulin.
