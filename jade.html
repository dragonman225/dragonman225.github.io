Highlights

Block-Card Duality: A Block can be expanded to become a Card.

Recent: Navigate between recently visited Blocks (=Cards).

💡

Currently the algorithm is just like how Web browser history behaves but without duplicated visits. Would LRU strategy be more useful?

Search-to-embed: Type a keyword to search all Blocks, click to expand (as Cards), drag out to embed (as Blocks).

Reactive Editing: Blocks that reference the same source changes synchronously.
