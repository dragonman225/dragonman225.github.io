Value Selection Rule

‚ö†Ô∏è
Since different product lines, different manufactures use different method to specify their products, sometimes comparison is unable to be done under same condition. However, I try to keep the condition difference as small as possible with the following rules :
  • SNR, Channel Seperation, THD+N : 24-bit, PCM mode, 1kHz input, Stereo mode, 2-V RMS & 0 dB output, A-weighted, minimum of typical
  • Pass-band ripple, Stop-band attenuation : Sharp roll-off or fastest roll-off
  • Current output devices only have Output Current specified, while Voltage output devices only have Output Voltage (Differential) specified.

Other Note

  • The color of Package name reflects friendliness for hand soldering. From the most friendly to the least friendly are : green, yellow, orange, red.
  • Unit Price are obtained from Digi-Key.
  • PCM1792A, PCM1795, PCM1796 are pin compatible.
  • PCM1794A, PCM1798 are pin compatible.
  • WM8740, WM8741 are pin compatible.
  • ESS Technology's 3 flagship 8-channel DAC : ES9018, ES9028, ES9038, which are specified at 130 dB+ SNR, but there are no public accessible datasheets.
  • Currently, it seems that ESS Technology, Asahi Kasei Microdevices, and Cirrus Logic are still developing and announcing new audio DACs with electrical performance surpassing previous products. TI's PCM1792A, PCM1794A and ADI's AD1955 have been around for more than 10 years.
  • Wolfson is now part of Cirrus Logic.
